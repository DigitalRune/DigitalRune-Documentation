Remarks

DigitalRune Mathematics uses column vectors whereas the XNA Framework uses row vectors. By converting a Matrix (XNA Framework) to Matrix44D (DigitalRune Mathematics) the matrix is automatically transposed.

This method is available only in the XNA-compatible build of the DigitalRune.Mathematics.dll.
