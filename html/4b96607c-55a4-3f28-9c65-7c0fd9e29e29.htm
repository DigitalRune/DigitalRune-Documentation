Remarks

This shape is a collection of points from which the convex hull is computed.

A ConvexHullOfPoints is similar to a ConvexPolyhedron except that the shape can be changed dynamically. Points in the ConvexHullOfPoints can be added or removed at runtime.

Use a ConvexHullOfPoints if the points in the shape need to be modified at runtime. Use a ConvexPolyhedron if the set of points is fixed and a high performance is required.
