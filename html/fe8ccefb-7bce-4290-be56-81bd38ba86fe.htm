DigitalRune Game

The DigitalRune Game library (DigitalRune.Game.dll) provides common functionality for game applications. The library is usually distributed as a part of other DigitalRune products.

This topic contains the following sections:

  • Features
  • Namespaces
  • Subtopics
Features

Here is an overview of features in this library:

  • A game object system supporting
    • Extendable and connectable properties,
    • Extendable events,
    • Templating,
    • Life-time management,
    • And more...

  • State machines supporting

    • Nested states,
    • Parallel states,
    • History states,
    • And more...
  • Clocks and timers to control game loop timing
Namespaces

Following namespaces belong to the DigitalRune Game library:

  • DigitalRune.Game
  • DigitalRune.Game.States
  • DigitalRune.Game.Timing
