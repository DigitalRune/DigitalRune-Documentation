Remarks

Notes to Inheritors: The parameters are guaranteed to be in a valid range - no parameter validation necessary.

If an exact mesh can be returned, this mesh should be generated. If the shape can only be approximated, the absolute distance error should be less than absoluteDistanceThreshold or at max iterationLimit iterations should be performed for iterative mesh generation algorithms.
