Xbox LIVE Avatars

DigitalRune Animation supports Xbox LIVE Avatars. Avatars can be animated, animations can be cross-faded, avatar ragdolls can be automatically created, and much more.

Xbox LIVE Avatars (Xbox 360 only)
 Caution

The Xbox LIVE Avatar support in DigitalRune Animation is newer and replaces the Xbox LIVE Avatar support in DigitalRune.Physics.Specialized (see Specialized Modules).

DigitalRune Animation also provides support for animating Xbox LIVE Avatars: The class AvatarPose describes the SkeletonPose of an avatar as well as its AvatarExpression (facial expression). AvatarPose implements IAvatarAnimation and therefore can be used directly with the XNA class for rendering.
