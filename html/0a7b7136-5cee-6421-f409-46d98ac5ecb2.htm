ScrollBar Class
Represents a control that provides a scroll bar that has a sliding Thumb whose position corresponds to a value and buttons to change the value.

Remarks
A ScrollBar has a sliding Thumb. Value defines the position of the thumb. ViewportSize defines the size of the thumb. The scroll bar also has two buttons that can be clicked to change the Value. The "empty" space between the thumb and the buttons can also be clicked (like a repeat button) to change the Value.
