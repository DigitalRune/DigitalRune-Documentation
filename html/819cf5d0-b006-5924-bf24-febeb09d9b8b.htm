Property Value

Type: Single
The specular intensity of the light. The default value is 1.
Remarks

Setting the specular intensity to NaN has a special meaning and is used to disable the specular contribution of this light. That means, if the intensity is NaN, the ImageBasedLight does not influence the specular light buffer of the scene.

Color and SpecularIntensity are separate properties so the values can be adjusted independently.
