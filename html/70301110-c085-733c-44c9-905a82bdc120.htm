Property Value

Type: Single
The scale of the second skeleton relative to the first skeleton. The default is 1.
Remarks

Translations that are mapped are multiplied with this scale factor. So if the second skeleton is about twice as large as the first skeleton, set this value to 2.

EstimateScale can be used to set this value to an automatically guessed scale factor.
