Remarks

This class is used to bind an EffectParameter of an Effect to a certain value. When the effect parameter represents a single value (such as Single, Vector3, Matrix, etc.) an EffectParameterBindingT should be used to bind the value. When the effect parameter represents an array of values an EffectParameterArrayBindingT needs to be used.

Cloning:
EffectParameterBindings need to be cloneable. The method Clone calls CreateInstanceCore and CloneCore(EffectParameterBinding) which are responsible for creating a clone of the current instance. Classes that derive from EffectParameterBinding need to provide the implementation for CreateInstanceCore and override CloneCore(EffectParameterBinding) if necessary.
