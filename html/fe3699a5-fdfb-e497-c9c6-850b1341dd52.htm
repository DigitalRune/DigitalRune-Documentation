Remarks
The number of particles in each ParticleSystem is limited (see MaxNumberOfParticles). If the limit is reached, no more new particles can be emitted until existing particles die. The ReserveParticleEffector monitors NumberOfActiveParticles and if less than Reserve particles are left, it kills the oldest particles (by setting their "NormalizedAge" to 1) in its OnUpdateParticles(TimeSpan, Int32, Int32) method. This effector ensures that there are always free particles ready to be emitted. Usually, this effector should be added as the last effector to the Effectors collection of the particle system because it should be updated after all other effectors.
