Remarks
This class is a collection of several helper methods to compute bounding volumes (see CreateBoundingShape(IListVector3F), ComputeBoundingBox(IListVector3F, Vector3F, Pose) and ComputeBoundingSphere(IEnumerableVector3F, Single, Vector3F)) and convex hulls (see CreateConvexHull(IEnumerableVector3F)). The other methods will not be needed in most situations. For general collision detection tasks use the types in the namespace DigitalRune.Geometry.Collisions.
