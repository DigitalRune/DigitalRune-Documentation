Property Value

Type: String
The name of the parameter that defines the bias velocity vector. (Parameter type: varying or uniform, value type: Vector3F)
The default value is "EmitterVelocity".
Remarks
This property needs to be set before the particle system is running. The particle effector might ignore any changes that occur while the particle system is running.
