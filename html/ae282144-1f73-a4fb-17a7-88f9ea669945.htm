Remarks

When a new GraphicsScreen is added to the collection it is inserted at the end of the list. The item at index 0 is considered the backmost graphics screen. The item at index (Count - 1) is considered the foremost graphics screen. The graphics service renders graphics screens back to front.

Note: The default enumerator (see GetEnumerator) iterates the items from back to front.
