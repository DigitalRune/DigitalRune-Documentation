Remarks

This method assumes that the matrix is a 3D scale/rotation/translation (SRT) matrix. Decompose(Vector3F, Matrix33F, Vector3F) returns when the matrix is not a valid SRT matrix. This is the case when two or more of the scale values are 0 or the last row of the matrix is something other than (0, 0, 0, 1).

DecomposeFast(Vector3F, Matrix33F, Vector3F) is a faster version of this method that can be used when it is certain that the matrix is a valid SRT matrix.

Examples
The following example shows how to compose the matrix scale, rotation, and translation components.
Copy
Matrix33F sr = rotation * Matrix33F.CreateScale(scale);
Matrix44F srt = new Matrix44F(sr, translation);
