Asset Creation

This section explains how to build custom assets (e.g. models, effects) and where to find pre-built assets (shaders, textures), which are required by DigitalRune Graphics at runtime.

Getting started

This documentation assumes that you are familiar with the XNA Content Pipeline and that you already know how to create a content project. If you are new to this topic, you may want to look at the XNA documentation first:

  • XNA Game Studio: Adding Art, Music, and Other Game Assets
  • MonoGame: Adding Art, Sounds, and More
