Remarks
When the parameter is less than the parameter of the first key or greater than the parameter of the last key, then the parameter is outside the regular curve. The outside behavior is determined by PreLoop and PostLoop. If the loop type is Oscillate the curve is mirrored after each loop cycle. This method returns if the parameter is outside and belongs to a curve loop which is mirrored to the regular curve.
