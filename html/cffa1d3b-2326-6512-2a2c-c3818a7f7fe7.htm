Remarks

This interpreter owns two dictionaries (see TechniqueDescriptions and ParameterDescriptions), which are empty by default. The dictionary keys are case-insensitive strings, e.g. "WorldViewProjection". The dictionary value is a delegate that returns a EffectTechniqueDescription or a EffectParameterDescription for the given string. New dictionary entries can be added to add support for new effect techniques and parameters.
