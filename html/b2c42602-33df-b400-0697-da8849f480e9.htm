Remarks
This effect needs two velocity buffers in the render context: (VelocityBuffer and LastVelocityBuffer (optional). In each buffer a velocity vector is stored per pixel. The velocity vector shows the movement (relative to texture space) that this pixel has moved. One velocity buffer should contain the velocity vectors of the current frame and the other optional one the velocity vectors of the last frame.
