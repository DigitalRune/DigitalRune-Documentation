Remarks

Notes to Inheritors: The method Clone calls CreateInstanceCore which is responsible for creating a new instance of the Light derived class and CloneCore(Light) to create a copy of the current instance. Classes that derive from Light need to implement CreateInstanceCore and CloneCore(Light).
