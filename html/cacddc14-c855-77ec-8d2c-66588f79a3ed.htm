Property Value

Type: String
The name of the parameter that defines the target value. (Parameter type: varying or uniform, value type: Single)
The default value is . If this parameter is missing, the target value is 1.
Remarks
This property needs to be set before the particle system is running. The particle effector might ignore any changes that occur while the particle system is running.
