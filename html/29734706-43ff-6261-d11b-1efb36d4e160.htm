Property Value

Type: Int32
The clipmap level where this terrain layer starts to fade out. The default value is MaxValue, which means that the terrain layer does not fade out.
Remarks

This is material parameter. Changing this property affects all terrain layers that share the same material.

See TerrainLayer for more details.
