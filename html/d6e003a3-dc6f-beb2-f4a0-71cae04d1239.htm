Remarks

Do not call this method directly (except when calling base in an implementation). This method is called internally by the Clone method whenever a new instance of the BasePartitionT is created.

Notes to Inheritors: Every BasePartitionT derived class must implement this method. A typical implementation is to simply call the default constructor and return the result.
