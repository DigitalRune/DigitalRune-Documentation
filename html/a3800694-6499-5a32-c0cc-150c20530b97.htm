Remarks

This method assumes that the matrix is a 3D scale/rotation/translation (SRT) matrix. Decompose(Vector3D, QuaternionD, Vector3D) returns when the matrix is not a valid SRT matrix. This is the case when two or more of the scale values are 0 or the last row of the matrix is something other than (0, 0, 0, 1).

DecomposeFast(Vector3D, QuaternionD, Vector3D) is a faster version of this method that can be used when it is certain that the matrix is a valid SRT matrix.

Examples
The following example shows how to compose the matrix scale, rotation, and translation components.
Copy
Matrix44D srt = Matrix44D.CreateTranslation(translation)
                * Matrix44D.CreateRotation(rotation)
                * Matrix44D.CreateScale(scale);
