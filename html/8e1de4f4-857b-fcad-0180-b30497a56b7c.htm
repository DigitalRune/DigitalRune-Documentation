Remarks

The WaterWavesRenderer is a SceneNodeRenderer that handles WaterNodes. If a WaterNode uses OceanWaves, the renderer creates the wave textures and stores the result in the properties DisplacementMap and NormalMap.

Render Target and Viewport:
This renderer changes the current render target of the graphics device because it uses the graphics device to render the wave maps into internal render targets. The render target and the viewport of the graphics device are undefined after rendering.
