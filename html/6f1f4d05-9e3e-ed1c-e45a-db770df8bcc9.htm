Remarks

This renderer requires the HiDef graphics profile. If the current graphics profile is Reach, Render(IListSceneNode, RenderContext, RenderOrder) throws a NotSupportedException.

The FigureRenderer is a scene node renderer which handles FigureNodes.

Render Target and Viewport:
This renderer renders into the current render target and viewport of the graphics device.
