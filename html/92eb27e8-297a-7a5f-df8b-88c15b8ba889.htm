The collision domain.
Remarks

This collision domain computes collisions for the rigid bodies.

This collision domain can be used to make collision queries (e.g. GetContacts(CollisionObject)) and it is allowed to add custom CollisionObjects to this collision domain.

The collision domain is automatically updated by the simulation. (See also SynchronizeCollisionDomain.)
