Remarks

The perspective projection can be set in several ways:

  • By setting the properties Left, Right, Bottom, Top, Near, and Far.
  • By setting assigning a projection matrix using Set(Matrix44F) or the property Inverse.
  • By calling on of the following methods: Set(Single, Single), Set(Single, Single, Single, Single), Set(Matrix44F), SetFieldOfView(Single, Single), SetFieldOfView(Single, Single, Single, Single), SetOffCenter(Single, Single, Single, Single), SetOffCenter(Single, Single, Single, Single, Single, Single),

The property ViewVolume defines the bounding shape of the projection which can be used for frustum culling. The shape is updated automatically when the properties of the projection change.

This class supports near plane clipping for portal and reflection rendering, see NearClipPlane.
