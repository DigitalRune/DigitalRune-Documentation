member Strength : float32 with get, set

Property Value

Type: Single
The strength factor. If this value is 0.0, the source image is not changed. If this value is 1.0, the colors of the source image are converted based on the lookup texture. If this value is between 0.0 and 1.0, a linear interpolation between the source image and the color graded image is returned. The default value is 1.0.
