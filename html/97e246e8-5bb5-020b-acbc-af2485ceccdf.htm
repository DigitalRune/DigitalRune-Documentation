Remarks

The ZipStorage does not directly read the ZIP archive from the OS file system. Instead, it opens the ZIP archive from another storage.

The PasswordCallback needs to be set to read encrypted ZIP archives. The ZipStorage supports ZipCrypto (all platforms) and AES-256 encryption (Windows only).

Thread-Safety: The ZipStorage is thread-safe. ZIP entries can be read simultaneously from one or multiple threads.
