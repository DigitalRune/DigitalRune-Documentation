Ragdoll Physics

The DigitalRune Physics library supports ragdoll physics. Ragdolls can be used for collision detection and procedural animations (such as real-time death animations). See Specialized Modules in DigitalRune Physics.

Samples

The CharacterAnimationSample and the AvatarSample (see Samples) demonstrate how to simulate ragdolls and how to blend procedural animations and key frame animations.
